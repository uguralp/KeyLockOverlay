# KeyLock Overlay

KeyLock Overlay is a lightweight Windows utility that displays a simple, unobtrusive on-screen overlay to show the current on/off status of your **Caps Lock** and **Num Lock** keys. 

## 🚀 Features (Current State)
- **Simple UI:** A clean and straightforward interface that currently just gets the job done.
- **Real-time Status:** Instantly see whether your Caps Lock or Num Lock is active without looking down at your keyboard.
- **Windows Support:** Built specifically for Windows desktop environments.

## 🚧 Future Development
*Please note:* This is currently a **simple starting point**. The project is actively planned to be developed further with more features, better customizability, and enhanced UI/UX in the future.
